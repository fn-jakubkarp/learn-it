# Examples

Illustrative only. The engine **never reads this directory** — it works against
`subjects/`. This is here so you can see what a learner's files look like for one
subject, end to end, before you start your own.

- `computer-networking/audit.md` — a filled-in knowledge audit.
- `computer-networking/roadmap.md` — the audit turned into a list of concepts.
- `computer-networking/notes.md` — conceptual notes the learner authored.
- `computer-networking/assessments/2026-06-15-apply.md` — one full assessment: the issued task, the learner's submission, and the evaluation result.

To start for real: `bun src/learn-it.ts init "<your subject>"`.
