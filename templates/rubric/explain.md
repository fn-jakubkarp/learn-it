# Rubric: explain (teach-back)

Score each dimension 0-20. Total 0-100. Pass at >= 70.

| Dimension | Full marks when… |
|-----------|------------------|
| Correctness | No factual errors or misconceptions. |
| Mechanism | Explains WHY/HOW it works, not just what it is. |
| Clarity | A novice would follow it; every term used is defined. |
| Completeness | Covers the core of the focus, no major gaps. |
| Independence | Done from memory — not paraphrased from open notes. |

`evaluate` records: total score, pass/fail, and the single weakest dimension to work on next.
