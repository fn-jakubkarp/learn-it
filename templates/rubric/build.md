# Rubric: build (ship a real artifact)

Score each dimension 0-20. Total 0-100. Pass at >= 70. **Expert tier requires a pass here.**

Before scoring, INTERROGATE the artifact: ask 2-3 probing questions ("why this approach?", "what breaks if X?"). A learner who can't defend their own work caps at 50 — building it isn't enough, you must own it.

| Dimension | Full marks when… |
|-----------|------------------|
| Works | The artifact demonstrably does what it claims. |
| Scope | Meets the stated acceptance criteria. |
| Sound choices | Decisions are justified and defended under questioning. |
| Integration | Ties multiple concepts together, end-to-end. |
| Originality | Genuinely the learner's work, not copied wholesale. |

`evaluate` records: total score, pass/fail, and the single weakest dimension to work on next.
