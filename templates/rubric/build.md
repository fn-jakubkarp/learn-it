# Rubric: build (ship a real artifact)

Score each dimension 0-20; total 0-100. Pass at >= 70. **Expert tier requires a pass here.**

Before scoring, INTERROGATE the artifact: ask 2-3 probing questions ("why this approach?", "what breaks if X?"). A learner who can't defend their own work caps at 50 — building it isn't enough, you must own it. This interrogation IS the refute-first step; score what survives it. When torn between two bands, pick the lower.

**Band, every dimension:** `0-5` absent or wrong · `6-12` partial, real gaps · `13-16` solid, minor gaps · `17-20` full (column below).

| Dimension | Full marks (17-20) | Zero marks (0-5) |
|-----------|--------------------|------------------|
| Works | The artifact demonstrably does what it claims. | Does not run / does not work. |
| Scope | Meets the stated acceptance criteria. | Misses the acceptance criteria. |
| Sound choices | Decisions are justified and defended under questioning. | Choices collapse on the first "why?". |
| Integration | Ties multiple concepts together, end-to-end. | One isolated trick, nothing joined. |
| Originality | Genuinely the learner's work, not copied wholesale. | Copied wholesale. |

`evaluate` records: total score, pass/fail, and the single weakest dimension to work on next.
