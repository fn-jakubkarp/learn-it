# Rubric: apply (solve a problem)

Score each dimension 0-20. Total 0-100. Pass at >= 70.

| Dimension | Full marks when… |
|-----------|------------------|
| Correctness | The solution actually works / is right. |
| Transfer | Applied to the NEW situation, not a memorized answer. |
| Method | Sound approach and valid reasoning, not luck. |
| Edge handling | Considers failure modes and boundary cases. |
| Efficiency | Reasonable, not needlessly convoluted. |

`evaluate` records: total score, pass/fail, and the single weakest dimension to work on next.
